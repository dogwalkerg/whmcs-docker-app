# whmcs-docker-app
一个运行在容器内的WHMCS镜像

支持WHMCS-V9.0.2版本源码容器    docker pull ghcr.io/dogwalkerg/whmcs-docker-app:latest
数据库    crpi-x68pqc6t5zexwzms.cn-guangzhou.personal.cr.aliyuncs.com/xianyuauto/whmcs-docker-mariadb:10.11

amd64 和 arm64  架构都支持。
